Background Images, Let's Go!

Properties

There are a few different background Properties:
background-image
This Property is the location of the image itself, for example:
h1{ background-image: url(' http:/website.com/img/background.jpg');}

or

h1 { background-image: url('image/photo.jpg') ;}
Background-color
This allows to add colour to a background. An example in CSS would like like this:
h1{ background-color: green;}
Background-repeat;
This is how the image will repeat. The value can be any of the following

background-repeat:repeat;

This repeats on the X and Y axis, and will essentialy tile the image across the entire

background-repeat: repeat-x;

This repeats the background on the X axis,or horizontally.

background-repeat: repeat-y;

This repeats the background on the Y axis,or vertically.

background-repeat: no-repeat;

This is show the image only once, without repeating.

Background-postion
This is the actual position of the background image within it's container. the values can be:
background-postion: top;
background-postion: right;
background-postion: bottom;
background-postion: left;
background-postion: center;

It can also be a percentage, length , or combination, like so:
background-postion: 50px 20%;

Shorthand

There's a quick way around background images,and you don't necessairly have to type code to set up your background images the way you want them.

Background
The "background" Shorthand can handle all in one!
background: green url('image/photo.jpg') top left no-repeat;
